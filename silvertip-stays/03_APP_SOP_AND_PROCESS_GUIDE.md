## Daily Workflows

### **Morning Routine (5 min)**

```
1. Login to app
2. Check Dashboard
3. Review key metrics:
   - Sales yesterday
   - Attendance today
   - Any alerts/issues
4. Plan your day based on data
```

### **During Day (Ongoing)**

```
- Restaurant staff: Observe sales flow
- Finance team: Track expenses, record payments
- HR team: Verify attendance (synced from Jibble)
- Manager: Monitor operations via Sales Reports
```

### **End of Day (15 min)**

```
1. Data Entry: Record manual expenses/sales not in Square
2. Verify Square sync completed
3. Check Jibble sync for attendance
4. Note any anomalies
5. Save records
```

### **Weekly Routine (30 min)**

```
1. Monday: Review previous week's data
2. Wednesday: Check sales trends
3. Thursday: Review payroll data
4. Friday: Analyze analytics, plan next week
```

### **Monthly Routine (1 hour)**

```
1. Close previous month
2. Review P&L statement
3. Compare to budget/targets
4. Analyze profitability by entity
5. Document insights
6. Plan adjustments for next month
```

## Data Management

### **How Data Flows Into the App**

```
Manual Entry
   ↓
Data Entry Forms
   ↓
Supabase Database
   ├── Square Sync (Auto, 2 AM daily)
   │   ├── Orders
   │   ├── Payments
   │   └── Sales Summary
   │
   └── Jibble Sync (Auto, 2:15 AM daily)
       ├── Employees
       ├── Attendance
       └── Payroll Data
   ↓
Real-time Display in App
```

### **Data Refresh Schedule**

| Source | Frequency | Time | Manual Refresh |
|--------|-----------|------|---|
| **Square** | Daily | 2:00 AM | Yes, via API button |
| **Jibble** | Daily | 2:15 AM | Yes, via API button |
| **Manual Entry** | Real-time | As entered | N/A |

### **Data Quality & Validation**

**System ensures**:
- ✅ No duplicate entries
- ✅ Timestamp accuracy
- ✅ Numeric validation (no negative unless expense)
- ✅ Mandatory field validation
- ✅ Data range checks

**You should verify**:
- ✅ Sales totals match cash count
- ✅ Attendance matches physical headcount
- ✅ Expenses are reasonable vs. history

### **Exporting Data**

From any table/report:
1. Click "Download" or "Export"
2. Choose format: CSV or Excel
3. File downloads to your computer

## Troubleshooting

### **"I don't see today's sales data"**

**Solution**:
1. Check if Square is connected
2. Wait until 2 AM for automatic sync
3. Or click "Sync Now" button
4. Verify Square has orders in the system
5. Check Sync Status table in admin

### **"Attendance data is missing"**

**Solution**:
1. Confirm Jibble is connected
2. Verify employees checked in/out in Jibble
3. Wait until 2:15 AM for sync
4. Or manually trigger sync
5. Check that employees are marked as "ACTIVE"

### **"Numbers don't add up"**

**Solution**:
1. Check for duplicate entries
2. Verify decimal places (e.g., 100.50 vs 10050)
3. Confirm date ranges match
4. Check if negative values were entered
5. Review transaction logs

### **"I can't access a module"**

**Solution**:
1. Verify your user role (check header)
2. Admin users: See all modules
3. Managers: See Operations modules only
4. Contact admin to change your role

### **"System is slow or lagging"**

**Solution**:
1. Refresh the page (F5 or Cmd+R)
2. Clear browser cache
3. Check internet connection
4. Try again in 5 minutes (system may be syncing)
5. Contact support if persists

### **"I see an error message"**

**Solution**:
1. Note the exact error message
2. Take a screenshot
3. Refresh and try again
4. Check Admin > Sync Status for errors
5. Contact support with screenshot and details
